The two readouts run 1.5 was built for

A new runner requires a new control: 07 anchors Claude Code the way 05 anchors Copilot CLI. That pairing isolates two effects run 1 couldn't see.

05 → 07 · +3.0 pts Runner effect, no harness anywhere. The Claude Code control wrote 77 tests to 05's 23, added an unprompted self-review commit that fixed two real bugs, and handled the naive-timestamp edge that crashes 05 — for +4 minutes of wall-clock (tests +1, process +0.5, efficiency −0.5).
07 → 06 · −8.0 pts gstack vs its own baseline, same runner. The harness improved no dimension. It bought the field's most hardened parser and 137 tests at 2.2× the control's wall-clock and ~3× its cost (docs −0.5, process −0.5, efficiency −3) — run 01's robustness-for-spend trade, replayed. Not a coincidence: ATV-StarterKit bundles gstack.
